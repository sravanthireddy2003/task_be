                              QUICK START (5 STEPS)
================================================================================

STEP 1: DATABASE SETUP
  $ mysql -u root -p your_database < database/audit_logs_optimization.sql

STEP 2: INSTALL DEPENDENCIES
  $ npm install uuid

STEP 3: DOWNLOAD ELK STACK (NO DOCKER)
  • Elasticsearch: elastic.co/downloads/elasticsearch
  • Kibana: elastic.co/downloads/kibana
  • Logstash: elastic.co/downloads/logstash
  • MySQL JDBC: dev.mysql.com/downloads/connector/j/

STEP 4: START SERVICES
  # Elasticsearch
  $ cd C:\elasticsearch
  $ bin\elasticsearch.bat
  
  # Kibana
  $ cd C:\kibana
  $ bin\kibana.bat
  
  # Logstash (edit config first!)
  $ cd C:\logstash
  $ bin\logstash.bat -f C:\path\to\logstash\logstash.conf

STEP 5: SETUP & VERIFY
  $ .\scripts\setup_elasticsearch.ps1
  $ node test_audit_system.js
  
  Open Kibana: http://localhost:5601
  Follow: kibana/dashboard_setup_guide.md

================================================================================
                              SUPPORT & TROUBLESHOOTING
================================================================================

No data in Elasticsearch?
  → Check Logstash logs
  → Verify MySQL connection
  → Check JDBC driver path
  → Review last_run timestamp

Dashboard empty?
  → Verify index pattern: audit-logs-*
  → Check time range (try Last 30 days)
  → Verify tenant_id filter not too restrictive

Slow logging performance?
  → Check MySQL indexes
  → Review circuit breaker status
  → Consider enabling queue (Bull/Redis)
  → Check Elasticsearch heap size
