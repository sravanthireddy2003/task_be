📋 STEP 4: INTEGRATE INTO YOUR APP (10 MIN)

──────────────────────────────────────────────────────────────────────────────
In src/app.js (add after existing middleware):
──────────────────────────────────────────────────────────────────────────────
const { auditMiddleware } = require('./middleware/auditLogger');
app.use(auditMiddleware());

──────────────────────────────────────────────────────────────────────────────
In src/controllers/AuthController.js (add at top):
──────────────────────────────────────────────────────────────────────────────
const auditLogger = require('../services/auditLogger');

// In login success:
await auditLogger.logAudit({
  action: 'LOGIN_SUCCESS',
  tenant_id: user.tenant_id,
  actor_id: user._id,
  module: 'Auth',
  entity: 'User',
  entity_id: user._id,
  ip_address: req.ip,
  user_agent: req.headers['user-agent'],
  details: { email: user.email, role: user.role }
});

📋 STEP 5: CONFIGURE LOGSTASH (15 MIN)

┌──────────────────────────────────────────────────────────────────────────┐
│ 1. Download MySQL JDBC Driver                                            │
│    https://dev.mysql.com/downloads/connector/j/                          │
│    → Platform Independent → ZIP                                          │

┌──────────────────────────────────────────────────────────────────────────┐
│ 2. Edit: logstash\logstash.conf                                          │
│                                                                           │
│    UPDATE:                                                                │
│    - jdbc_connection_string (database name)                              │
│    - jdbc_user (your MySQL username)                                     │
│    - jdbc_password (your MySQL password)                                 │
│    - jdbc_driver_library (path to JAR file)                              │
│    - elasticsearch hosts (http or https)                                 │
└──────────────────────────────────────────────────────────────────────────┘

┌──────────────────────────────────────────────────────────────────────────┐
│ 3. Start Logstash                                                         │
│                                                                           │
│    cd C:\logstash                                                         │

│                                                                           │
│    Watch for: "Successfully connected to database"                       │
